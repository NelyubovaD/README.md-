# Проект: «Накладные ресницы»

- **Группа:** 02141

## 2. Тематика сайта, название
**Тематика:** накладные ресницы (виды, материалы, бренды, техника наклеивания, уход, сравнения, где купить).  
**Название сайта:** *Lash Guide — всё о накладных ресницах*  
**Краткое описание:** Информационно-практический сайт для тех, кто хочет разобраться в накладных ресницах. Подходит новичкам (как выбрать и наклеить), мастерам (обзоры материалов, сравнение клеев) и обычным девушкам, которые хотят быстро сделать выразительный взгляд. Без воды: схемы, рейтинги, пошаговые инструкции, отзывы на популярные бренды.

## 3. Цели и задачи сайта

**Цели:**
- Дать системные знания о накладных ресницах: от выбора до носки.
- Помочь избежать ошибок (клеить криво, использовать плохой клей, повредить родные ресницы).
- Стать удобным справочником для мастеров и клиентов.

**Задачи и их реализация на сайте:**
| Задача | Как решается |
|--------|---------------|
| Как выбрать тип ресниц (пучки, лента, 3D, объёмные) | Таблица сравнения: материал, длина, изгиб, эффект |
| Техника наклеивания | Видео-гайды с открытых источников + текстовые шаги с фото |
| Какой клей лучше | Обзоры клеев (чёрный, прозрачный, гипоаллергенный) с оценками |
| Уход за накладными ресницами | Памятка: как снимать, чистить, сколько можно носить |
| Где купить дёшево и качественно | Ссылки на проверенные магазины (Ozon, Wildberries, Ali) с фильтром |

## 4. Целевая аудитория

**Кто:**
- Девушки 14–40 лет, которые хотят пользоваться накладными ресницами (не наращивание, а именно накладные).
- Студентки, офисные сотрудницы, бьюти-блогеры.
- Начинающие мастера по ресницам (для сравнения материалов).
- Покупатели с бюджетом от 200 до 2000 рублей за пару.

**Потребности:**
- Простые и понятные инструкции.
- Честные отзывы без рекламы дорогих брендов.
- Решения для разных форм глаз (азиатских, круглых, глубоко посаженных).
- Советы «на каждый день»: как быстро наклеить за 5 минут.

**Особенности для дизайна:**
- Светлый, нежный дизайн (пастельные тона, фото глаз с ресницами).
- Адаптация под мобильные (80% будут с телефона).
- Удобный поиск по типам ресниц (например, фильтр: длина 10-12 мм, изгиб C, натуральный эффект).

## 5. Содержание и структура сайта

**Разделы (главное меню):**

1. **Главная** — приветствие, популярные статьи, топ-5 ресниц месяца.
2. **Виды ресниц** — ленточные, пучковые, объёмные, 3D, цветные, нижние ресницы.
3. **Как клеить** — пошаговые уроки (с обычным клеем, с подводкой-клеем, без зеркала).
4. **Обзоры брендов** — Kiss, Ardell, Vivienne, 1DAY, Tety, локальные бренды.
5. **Уход и хранение** — как продлить жизнь накладным ресницам.
6. **Где купить** — ссылки на магазины с ценами, акционные предложения.
7. **Отзывы и вопросы** — комментарии посетителей, FAQ (можно ли спать в ресницах?).
8. **Контакты** — предложения.
